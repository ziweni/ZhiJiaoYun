# 职教云自助刷课程序

> *说明*：**本程序仅供学习和参考，禁止用于商业或违法犯罪用途！**

*环境/依赖*

* Python 3.8.2

* Lib: alive-progress

> 运行前记得去 **main.py** 文件把 *obj.login_m("user", "pass")* 替换成你的账号和密码哟～
> 仅第一次登陆会使用账号和密码, 拿到 Cookies 后就可以留空啦！

> ⚠️  运行前请先确认环境和依赖是否正确！

*运行*

```shell
git clone https://github.com/shanling2016/ZhiJiaoYun && cd ZhiJiaoYun/
```

```shell
python3 main.py
```
